# 🎬 MovieVerse

MovieVerse is a modern movie search web application built using HTML, CSS, and JavaScript. It allows users to search for any movie and instantly view its details using the OMDb API.

## ✨ Features

- 🔍 Search any movie
- ⭐ IMDb Rating
- 🎬 Movie Poster
- 📅 Release Year
- 🎭 Genre
- 🎬 Director
- 👨‍🎤 Actors
- 🌍 Country
- ❌ Movie Not Found UI
- 📱 Fully Responsive
- ✨ Modern Glassmorphism UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- OMDb API
